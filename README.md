# MegaProject
**Юдин Станислав Анатольевич**
Графическое приложение для генерации надёжных случайных паролей с настраиваемыми параметрами. Позволяет выбирать длину, типы символов (заглавные, строчные, цифры, спецсимволы), просматривать историю генераций и автоматически сохранять её в формате JSON.

Возможности
- Ползунок регулировки длины пароля (4–128 символов)
- Гибкий выбор набора символов через чекбоксы
- Валидация ввода (мин/макс длина, обязательный выбор хотя бы одного набора)
- Копирование результата в буфер обмена одним кликом
- Таблица истории с датой, временем и параметрами генерации
- Автоматическое сохранение/загрузка истории из `password_history.json`

Зависимости
- Python 3.6+
- Стандартные библиотеки: `tkinter`, `random`, `json`, `string`, `datetime`
*(Дополнительные пакеты не требуются)*

Запуск
```bash
python main.py
